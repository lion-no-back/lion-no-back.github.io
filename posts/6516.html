具体过程

表层原理： 通过徽章生成网站Shields来制作静态徽章

需知： 要制作像模像样的徽章，就要明白并运用徽章制作规则

起手式

首先来到Shields首页，然后点击Get started

接着在实验台根据提示来制作第一枚徽章吧

结束式

具体过程就是输入相关信息后，点击Execute即可得到徽标的URL，可以用img标签引用。如果仅是这样肯定还不够个性化，所以它还提供了可选的参数来生成更多样式

建议用Chrome自带翻译功能进行翻译，不然中国本土人可能不怎么习惯，这里简单整理下

可选参数 说明
badgeContent 标签、（可选）消息和颜色。用破折号分隔
style 有扁平（默认）、扁平方形、塑料、徽章、社交这五种样式
logo 可以在左侧标签前插入图标，其图标可以在simpleicons上根据图标标题找到
logoColor 徽标的颜色
label 覆盖默认的左侧文本
labelColor 左侧部分的背景颜色
color 右侧部分的背景颜色
cacheSeconds HTTP 缓存生命周期
link 单击徽章后要跳转的超链接。这只适用于将徽章集成到<object>HTML 标记中，而不是<img>标记或标记语言中

可以使用 Hex、rgb、rgba、hsl、hsla 和 css 命名颜色
